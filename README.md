# 📚 Sistema de Lista de Notas em Java

Projeto acadêmico desenvolvido em **Java** utilizando **interface gráfica com Swing** e estrutura de dados do tipo **Pilha Estática (TAD)**.

---

## 🎯 Objetivo

O sistema foi desenvolvido para realizar o cadastro de alunos e gerenciamento de notas escolares, aplicando conceitos de:

- Estrutura de Dados
- Pilha Estática
- Programação Orientada a Objetos
- Interface Gráfica com Swing
- Manipulação de dados em Java

---

## ⚙️ Funcionalidades

✅ Cadastro de alunos  
✅ Inserção das notas das disciplinas:
- Matemática
- Português
- Geografia
- História
- Ciências

✅ Armazenamento dos dados em pilha estática  
✅ Cálculo automático da média geral  
✅ Verificação automática da situação do aluno:
- ✔️ Aprovado
- ⚠️ Recuperação
- ❌ Reprovado

✅ Exibição completa das informações dos alunos

---

## 📋 Regras de Aprovação

- O aluno é aprovado quando possui nota igual ou superior a **6** em todas as disciplinas.
- Caso possua alguma nota abaixo de 6, mas média geral igual ou superior a 6, o aluno fica em **recuperação**.
- Caso contrário, o aluno é considerado **reprovado**.
